Aproximadamente 2 minutos para completar una factura completa en el sistema.

Pasos del Proceso de Facturación

Sigue estos pasos secuenciales para registrar una venta exitosa en el sistema Sinnut ERP:

1
Acceder al Módulo
Accede al módulo Gestión de ventas y haz clic en Facturas propias para iniciar el proceso de facturación.
2
Validar Cliente
Haz clic en el botón + arriba a la derecha e ingresa el tipo y número de documento. Si el cliente no está registrado, el sistema te permitirá crear un nuevo registro.
3
Agregar Productos
Ingresa el o los SKU's de los productos y oprime el botón Agregar para incorporarlos a la factura. Puedes filtrar por tipo, estado o familia si es necesario.
4
Revisar y Aplicar Descuentos
Revisa el total de la venta y aplica un código de descuento si corresponde. Verifica que todos los datos sean correctos antes de proceder.
5
Procesar Pago
Selecciona el medio de pago apropiado y haz clic en Causar y pagar. El sistema generará la factura automáticamente y enviará el PDF correspondiente.
Consejo Importante

Asegúrate que el SKU ingresado sea el correcto para el producto que se quiere vender y que se encuentre disponible en inventario para evitar errores al momento de facturar.

Ejercicio Práctico Sugerido

Realiza una factura completa siguiendo todos los pasos: valida un cliente existente, agrega al menos dos productos diferentes, aplica un descuento y procesa el pago completo.
