Consultar Plan Separe

Aprende a buscar y consultar planes separe existentes para verificar pagos, realizar abonos e imprimir documentos en el sistema Sinnut ERP

30 segundos
4 Pasos
Consulta Rápida

¿Qué es la Consulta de Plan Separe?

La consulta de plan separe te permite buscar y visualizar planes separe existentes para verificar su estado, realizar abonos, revisar pagos realizados o continuar con el proceso de facturación.

Objetivo: Localizar rápidamente cualquier plan separe utilizando diferentes criterios de búsqueda y acceder a toda la información relacionada con el mismo.

Pasos del Proceso

Sigue estos pasos para consultar eficientemente cualquier plan separe

1
Acceder al Módulo
Dirígete al módulo Gestión de Ventas y selecciona Plan Separe. Allí verás un listado completo de todos los planes separe creados en el sistema.
Navegación: Gestión de Ventas → Plan Separe → Ver listado completo
2
Utilizar Filtros de Búsqueda
Usa la barra de búsqueda para filtrar los planes utilizando diferentes criterios como número de secuencia o documento del cliente.
Tip de búsqueda: Puedes buscar por número de documento, nombre del cliente o número de secuencia del plan para encontrar rápidamente lo que necesitas.
3
Ver Detalles del Plan
Haz clic sobre la fila del plan separe que deseas consultar para ver información detallada: estado actual, abonos realizados, saldo pendiente y documentos relacionados.
Información disponible: Estado del plan, historial de abonos, saldo pendiente, fechas importantes y documentos generados.
4
Acceder a Documentos
Si el plan separe está cancelado o facturado, podrás ver y descargar todos los documentos generados: facturas, recibos de abono y comprobantes de pago.
Documentos disponibles: Facturas de venta, recibos de abono, comprobantes de pago y reportes del plan separe.
🎯

¡Ahora es tu turno!

Practica consultando diferentes planes separe en el sistema para familiarizarte con la interfaz y las opciones disponibles.

Ejercicio sugerido: Busca un plan separe utilizando el documento del cliente y revisa su estado actual y historial de abonos.
