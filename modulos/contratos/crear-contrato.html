¿Qué es Crear un Contrato?

Este es el paso fundamental para iniciar una relación comercial con el cliente en la plataforma SINNUT. Aquí aprenderás a registrar un contrato, seleccionar el tipo de operación, agregar joyas y finalizar correctamente el proceso.

Objetivo: Dominar el proceso completo de creación de contratos, incluyendo registro de clientes, selección de categorías, adición de productos y finalización del proceso.

Tiempo estimado de operación: 6 minutos

Ajustar Precio de Contratación de Joyas

En algunos casos, al momento de crear el contrato es necesario modificar manualmente el precio de contratación según la negociación con el cliente o el tipo de joya.

Tiempo estimado de operación: 2:30 minutos

Agregar Materiales Complementarios

Cuando una joya incluye elementos como piedras, hilo, neopreno u otros materiales complementarios, es fundamental detallarlos correctamente durante la creación del contrato para evitar inconsistencias en el inventario y asegurar trazabilidad.

Tiempo estimado de operación: 1:30 minutos

Pasos para Crear un Contrato

Sigue estos pasos para crear un contrato exitoso en el sistema

1
Acceder al Módulo
Ingresa al módulo Gestión de contratos desde el menú principal del sistema.
Consejo: Asegúrate de tener los permisos necesarios para crear contratos.
2
Iniciar Creación
Haz clic en el botón "+" en la parte superior derecha para iniciar la creación del contrato.
Importante: Verifica que tengas todos los documentos del cliente antes de proceder.
3
Validar Cliente
Valida si el cliente existe. Si no, regístralo ingresando su tipo y número de documento.
Tip: Si el sistema no reconoce al cliente, verifica bien el tipo y número de documento. En caso de crear un nuevo cliente, asegúrate de completar todos los campos requeridos.
4
Seleccionar Categoría
Selecciona la categoría del contrato (oro, plata, artículo, acero) y el tipo (con pacto de retroventa o compra directa).
Buena práctica: Revisa cuidadosamente el tipo de contrato según la negociación establecida con el cliente.
5
Agregar Productos
Agrega productos al contrato ingresando el tipo fabricación, estado, familia, calidad y peso. El sistema calculará el valor automáticamente.
Recordatorio: Si necesitas contratos con porcentajes especiales, recuerda hacerlo durante la creación del mismo.
6
Documentar Producto
Toma la fotografía del producto, ingresa el código de la bolsa de seguridad y continúa.
Importante: Asegúrate de que la fotografía sea clara y muestre todos los detalles relevantes del producto.
7
Finalizar Proceso
En la vista de resumen, revisa el contrato, aplica un código de descuento si aplica, y finaliza el proceso haciendo clic en Terminar y luego Causar y pagar.
Verificación final: Revisa todos los datos antes de confirmar la transacción para evitar errores.
